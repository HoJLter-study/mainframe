## Все заметки с #history
```query
tag:#history line:(200-300)
```
```javascript
// Простой скрипт для добавления ссылки
function addToMOC(noteName) {
    const mocContent = app.vault.getAbstractFileByPath("MOC History.md");
    const newContent = mocContent + `\n[[${noteName}]]`;
    app.vault.modify(mocContent, newContent);
}
```
